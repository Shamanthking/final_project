
# Microservices Project

## Services
- User Service
- Student Service
- Course Service
- College Service

## How to Run
1. Install dependencies:
   npm install

2. Run services individually:
   npm run dev

## APIs

### GET
- /users
- /students
- /courses
- /colleges

### POST
- /users
- /students
- /courses
- /colleges

All data stored in-memory (arrays).
